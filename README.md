# next-msw-rails-sample

学習用の日報アプリ

## Setup

```
cd frontend && yarn
cd ../api && bundle install
```

## Development

- **frontend**: http://localhost:8080
- **api**: http://localhost:3000

## Tech Stack

### Frontend

- Next15
- NextAuth
- Tailwind, daisyUI
- Mock Service Worker

### Backend

- rails
- rspec
- devise-token-auth
